# NBA Analytics Platform 🏀

I like basketball, and I also enjoy learning coding.

During the 2026 NBA Finals, I wanted to compare the Knicks and the Spurs and see which team had a better chance to win the championship.

Then one idea suddenly came to my mind:

**"Why not build something using real NBA data?"**

I love using data to discover things that are not easy to see. Sometimes numbers can tell a different story from what we see on the court.

So I started this project.

Right now, it can download NBA player statistics, save them into CSV files, and generate a simple player ranking report.

This is only my first step. As I learn more programming, I want to keep making this project better and better.

## What it does

- Download NBA player statistics
- Save the data into CSV files
- Rank NBA players
- Generate a Top 10 report

## Built With

- Python
- Pandas
- nba_api
- Tabulate

## How to Run

Install the packages:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
